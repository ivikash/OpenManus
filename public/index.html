Browser Use Server

Status: Running

Port: 3001

This is the backend server for the Browser Use Demo application. It provides WebSocket connections and API endpoints for browser automation.

Available Endpoints

Health Check

GET /api/health

Returns the current status of the server.

WebSocket Connection

Connect to ws://localhost:3001 for real-time communication.

Available events:

  • prompt:submit - Send automation instructions
  • automation:stop - Stop current automation
  • automation:log - Receive automation logs
  • automation:screenshot - Receive browser screenshots
  • automation:complete - Automation completion notification

For more information, see the project documentation.
